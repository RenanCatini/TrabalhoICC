-Como utilizar o Git-

Esse site irá ensinar como usar o sistema de controle
do Git e o repositório Github.

Introdução

Git é uma ferramenta que ajuda a organizar e controlar as mudanças no código de um projeto.
Ela permite que várias pessoas trabalhem juntas sem bagunçar nada, e ainda guarda um histórico
de todas as mudanças, facilitando voltar atrás se algo der errado.

Git

Github

Para configurar o GitHub, siga estes passos:

  1. Crie uma conta no GitHub.
  2. Crie um novo repositório.
  3. Clone o repositório em seu computador local usando git clone.
  4. Adicione e modifique arquivos no repositório local.
  5. Adicione arquivos ao índice do Git usando git add.
  6. Confirme as alterações com git commit.
  7. Envie as alterações para o repositório remoto no GitHub com git push.
